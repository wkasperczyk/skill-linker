# Lookup: Tool Default Paths

**Purpose**: Quick reference for default tool paths in skill-linker

| Path Type | Default Value |
|-----------|---------------|
| Skills Repository | `~/.config/skill-linker/skills` |
| OpenCode Skills | `~/.opencode/skills` |
| Config File | `~/.config/skill-linker/config.json` |

**Config Structure**:
```json
{
  "SkillsRepositoryPath": "~/.config/skill-linker/skills",
  "OpencodeSkillsPath": "~/.opencode/skills",
  "ConfigFilePath": "~/.config/skill-linker/config.json"
}
```
